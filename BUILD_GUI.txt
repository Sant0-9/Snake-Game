                    BUILD YOUR SNAKE GAME AS A REAL APP!
================================================================================

You want a proper GUI application instead of terminal? HERE IT IS!

================================================================================
STEP 1: INSTALL SDL2 (ONE-TIME SETUP)
================================================================================

Run this command:
    ./install_sdl.sh

Or manually:
    sudo apt-get update
    sudo apt-get install -y libsdl2-dev libsdl2-ttf-dev libsdl2-mixer-dev libsdl2-image-dev

================================================================================
STEP 2: BUILD THE GUI VERSION
================================================================================

    make clean
    make snake_gui

================================================================================
STEP 3: RUN YOUR GUI APP!
================================================================================

    ./snake_gui

Or simply:
    make run-gui

TROUBLESHOOTING:
----------------
Problem: "SDL2 not found"
Solution: Run ./install_sdl.sh

Problem: Compilation errors
Solution: sudo apt-get install build-essential libsdl2-dev libsdl2-ttf-dev
